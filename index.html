Complex Numbers Quiz

Q1: La partie réelle du nombre complexe z = (2 + i)2 est :

  • A) 2
  • B) 4
  • C) 3

Correct Answer: C) 3

Q2: La partie imaginaire du nombre complexe z = (1 - i)2 est :

  • A) -2i
  • B) 0
  • C) -2

Correct Answer: C) -2

Q3: Le module du nombre complexe z = 4 + 3i est égal à :

  • A) 7
  • B) 7
  • C) 5

Correct Answer: C) 5

Q4: Un argument du nombre complexe z = 2 - 2i est égal à :

  • A) 2π/4
  • B) -4π/4
  • C) 3π/4

Correct Answer: C) 3π/4

Q5: Si z = 2 - 5i, alors :

  • A) z = 2 + 5i
  • B) z = -2 + 5i
  • C) z = -2 - 5i

Correct Answer: C) z = -2 - 5i

Q6: Le conjugué de z = 1 + 2i est :

  • A) 1 + 2i
  • B) 1 - 2i
  • C) -1 - 2i

Correct Answer: B) 1 - 2i

Q7: La partie réelle de z = -1 - i est :

  • A) -1
  • B) -i
  • C) 1

Correct Answer: A) -1

Q8: La partie imaginaire de z = 2 + 3i est :

  • A) 2
  • B) 3
  • C) i

Correct Answer: B) 3

Q9: Le produit de z = (1 + i) par z = (1 - i) est :

  • A) 1
  • B) -1
  • C) 2

Correct Answer: C) 2

Q10: Le module de z = 3 + 4i est :

  • A) 5
  • B) 6
  • C) 7

Correct Answer: A) 5

Q11: La partie imaginaire du produit de z = (2 + i) par z = (1 - i) est :

  • A) 4
  • B) -1
  • C) 3

Correct Answer: B) -1

Q12: Si z = 4 + 3i, alors :

  • A) |z| = 4
  • B) |z| = 3
  • C) |z| = 5

Correct Answer: C) |z| = 5

Q13: L'inverse de z = 1 + i est :

  • A) 1 - i
  • B) -1 - i
  • C) (1/2) - (i/2)

Correct Answer: C) (1/2) - (i/2)

Q14: Le carré de z = 1 + i est :

  • A) 2i
  • B) -2i
  • C) -2

Correct Answer: C) -2
